# 🚀 Resume Builder Pro

A modern and user-friendly Resume Builder web application that helps users create professional ATS-friendly resumes in minutes. Fill in your details, preview your resume instantly, and download it as a PDF.

## ✨ Features

* 📝 Easy Resume Creation
* 👀 Live Resume Preview
* 📄 ATS-Friendly Resume Template
* 🎓 Education Section
* 💻 Technical Skills Section
* 🚀 Projects Section
* 📜 Certifications Section
* 🎯 Hobbies & Interests
* 🏆 Extracurricular Activities
* 📱 Responsive Design
* 💾 Local Storage Support
* 📥 PDF Download Functionality

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* HTML2PDF.js

```text

```

## 📋 How to Use

1. Enter your personal details.
2. Add education, skills, projects, and certifications.
3. Preview your resume in real time.
4. Download your resume as a PDF.
5. Share your professional resume with recruiters.
